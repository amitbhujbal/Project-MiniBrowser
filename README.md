# Project-MiniBrowser
A lightweight in-app browser built using WKWebView.

## Demo

![Demo.](https://amitbhujbal.com/images/demo-Project-MiniBrowser.gif "Demo")

## Features

- Load website via URL input
- Automatic https handling
- Back / Forward navigation
- Reload support
- Basic toolbar controls

## Technical Details
- Built with Swift
- Uses UIKit and WebKit (WKWebView)
- Supports iOS 15+
- Developed using Xcode 15+
